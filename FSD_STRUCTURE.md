# Структура проекта в архитектуре Feature-Sliced Design (FSD)

## Общая структура

```

├── app/                      # App layer - инициализация приложения
│   ├── providers/            # Провайдеры приложения
│   ├── routes/               # Настройка маршрутизации
│   └── App.tsx               # Главный компонент приложения

│   ├── login/                # Страница входа
│   │   ├── ui/               # UI компоненты страницы
│   │   ├── index.tsx         # Главный файл страницы
│   │   └── model/            # Логика страницы
│   └── products/             # Страница товаров
│       ├── ui/               # UI компоненты страницы
│       ├── index.tsx         # Главный файл страницы
│       └── model/            # Логика страницы
├── widgets/                  # Widgets layer - составные компоненты
│   ├── product-table/        # Таблица товаров
│   │   ├── ui/               # UI компоненты виджета
│   │   ├── model/            # Логика виджета
│   │   └── index.ts          # Экспорты виджета

│   │   ├── ui/               # UI компоненты виджета
│   │   ├── model/            # Логика виджета
│   │   └── index.ts          # Экспорты виджета
│   └── auth-widget/          # Виджет авторизации
│       ├── ui/               # UI компоненты виджета
│       ├── model/            # Логика виджета
│       └── index.ts          # Экспорты виджета
├── features/                 # Features layer - бизнес-функции
│   ├── add-product/          # Функция добавления товара
│   │   ├── ui/               # UI компоненты функции
│   │   ├── model/            # Логика функции

│   ├── auth/                 # Функция авторизации

│       └── index.ts          # Экспорты функции
├── entities/                 # Entities layer - бизнес-сущности
│   ├── product/              # Сущность товара
```

## Описание слоёв

### app (Слой приложения)
- Инициализация приложения
- Глобальные провайдеры
- Настройка маршрутизации
- Глобальные стили

### pages (Слой страниц)
- Страницы приложения
- Композиция виджетов и фич
- Логика конкретной страницы

### widgets (Слой виджетов)
- Составные компоненты
- Композиция фич и сущностей
- Переиспользуемые блоки UI

### features (Слой фич)
- Бизнес-функции
- Логика конкретной функции
- UI для выполнения функции

### entities (Слой сущностей)
- Бизнес-сущности
- Типы данных
- API для работы с сущностями
- UI для отображения сущности

### shared (Слой общих ресурсов)
- Переиспользуемые компоненты
- Утилиты
- Константы
- Типы

### segments (Слой сегментов)
- Группировка связанных фич
- Крупные блоки приложения

## Правила импорта

Импорты возможны только от нижних слоёв к верхним:
- shared ← entities ← features ← widgets ← pages ← app

Запрещены импорты:
- От верхних слоёв к нижним
- Между элементами одного слоя
- Пропуск слоёв при импорте (кроме shared)
